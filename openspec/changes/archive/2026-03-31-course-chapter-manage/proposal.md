## Why

为了完善课程管理功能，不仅需要对课程基本信息进行维护，还需要构建课程的具体教学大纲（章节与课时）。本变更旨在“课程编辑”页面中增加一个“章节课时” Tab，允许用户通过可视化的界面管理课程大纲，包括创建章节、创建/编辑课时内容，并支持选择本地资源库中的教学视频，提升课程内容编排的效率与直观性。

## What Changes

- 在课程详情/编辑页面增加“章节课时” Tab 选项卡。
- 增加大纲构建区，支持添加新章节、在章节下添加/编辑课时。
- **支持在章节间拖拽排序章节，以及在章节内部拖拽排序课时。**
- 课时列表右侧展示当前挂载状态（已显示关联视频名称或显示“未挂载”）。
- 实现课时编辑的侧边抽屉/弹层，允许修改课时名称、简介，并从资源库选择教学视频。
- 使用 JSON 格式完成前端交互的数据状态管理，并预留后端对接使用的 API 接口方法。

## Capabilities

### New Capabilities
- `course-outline-manage`: 课程大纲（章节与课时）的可视化构建、编辑，以及视频资源绑定的能力。

### Modified Capabilities

## 非目标 (Non-goals)

- 本次变更**仅实现前端交互**，真实的后端接口（如保存大纲、处理分片上传等）暂不在本次工作范围内，仅预留占位方法。
- 不涉及从零搭建新的资源上传组件，主要聚焦于选择与绑定逻辑（如果已有资源库则复用）。

## Impact

- 影响 `src/pages/CourseManage/CourseDetail` 及其相关的子组件（新增 `ChapterLessonTab` 等）。
- 需扩展课程详情的状态或引入专门的 context/store 来维护 chapters/lessons 的 JSON 树形数据。
- 预留相关的 API Service 定义。
